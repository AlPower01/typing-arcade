# Asset Pipeline

## Backgrounds
Place backgrounds in:
- assets/backgrounds/city/
- assets/backgrounds/desert/
- assets/backgrounds/lab/

Register each background ID in:
- src/content/manifests/backgrounds.manifest.json

## Characters
Place character sprites in:
- assets/characters/player/
- assets/characters/enemies/
- assets/characters/bosses/

Register each logical ID in:
- src/content/manifests/characters.manifest.json

## Effects
Place bullets, hits, and explosions in:
- assets/effects/bullets/
- assets/effects/hits/
- assets/effects/explosions/

Register each logical ID in:
- src/content/manifests/effects.manifest.json

## Audio
Place placeholder or final audio in:
- assets/audio/music/
- assets/audio/sfx/

Register audio IDs in:
- src/content/manifests/audio.manifest.json
