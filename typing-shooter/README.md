# Typing Shooter Starter Pack

This starter pack gives you a clean data-driven structure for a side-scrolling typing shooter:
- sample levels
- enemy and boss configs
- word lists
- manifests
- placeholder sprite assets
- instance maps
- architecture docs

## Included
- 3 sample levels
- 6 enemy definitions
- 3 boss definitions
- placeholder SVG backgrounds
- placeholder SVG character, bullet, hit, and explosion assets
- `instance-map.json` showing exactly what spawns where

## Suggested next step
Use this repo as the content and structure spec inside Codex, then ask Codex to scaffold the game runtime around these files.
